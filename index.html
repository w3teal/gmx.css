    GMX.css implements the Material 3 design system in Vanilla CSS/JS, from the components to the animations to the theming.



    Getting started
    <link rel="stylesheet" href="https://unpkg.com/gmx.css">
<script src="https://unpkg.com/iconify-icon"></script>

    Or install it

    npm install gmx.css
npm install iconify-icon


    Still in development, some features may not work properly and still many breaking changes. Nothing is guaranteed.

    Fast

    GMX.css is an rewritten Material Design-based CSS framework, with an smallest size and good code quality.

    Robust

    Even in some edge cases like being printed, used in high contrast mode, or with JS off, most of GMX.css is still usable.


    Pure

    The code is written just with pure CSS and dont limited to one framework, so you can reusable it with HTML5 or your favorite framework.


    Micro

    Any Material Design-based framework was really heavy at size, GMX.css is built efficient and tiny as possible, makes it 10x smaller than any framework.

    Extra JS

    Slider: 
    function updateSlider(slider) {
    const value = slider.value;
    const percent = value + '%';
    slider.parentElement.style.setProperty('--percent', percent);
    const tooltip = slider.parentElement.querySelector('.tooltip');
    tooltip.textContent = value;
}
    Example usage: 
    <label class="slider" style="--percent: 53%">
    <input type="range" value="53" min="0" max="100" step="1" oninput="updateSlider(this)">
    <span class="track"></span>
    <span class="thumb"></span>
    <span class="tooltip">53</span>
</label>

    document.onkeydown = function (e) {
    if (e.keyCode == 27) {
        var mods = document.querySelectorAll('.dialog-trigger+[type=checkbox]');

            mod.checked = false;
        });
    }
